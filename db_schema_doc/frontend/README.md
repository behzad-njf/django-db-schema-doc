# ERD frontend (Vite + React Flow)

## Architecture

```
src/
  types/           # Shared TypeScript models (graph payload, table detail)
  lib/             # Pure functions (parse, filter, neighbors) — no React
  hooks/           # Stateful UI logic (useErdGraph, useDarkMode)
  components/
    landing/       # Full-screen welcome + file picker
    toolbar/       # Top bar controls (ERD mode)
    erd/           # React Flow canvas + ErdView layout
    panel/         # Table detail side panel
    nodes/         # Custom React Flow node types
  styles/          # Global CSS variables and layout
  App.tsx          # Composition root only
```

- **Data flow:** User picks `graph.json` on landing → `parseGraphPayload` → `useErdGraph` → `ErdView` (canvas + detail panel)
- **No auto-fetch** — graph loads only from the file picker
- **Routes:** `#/` (landing), `#/erd` (explorer). Graph is stored in `sessionStorage` so refresh on `#/erd` keeps your diagram.
- **GitHub Pages:** https://behzad-njf.github.io/django-db-schema-doc/ (build with `VITE_BASE_PATH=/django-db-schema-doc/ npm run build`)
- **Table details:** Embedded in each node's `data.table` (generated by `export_graph_dict` on the backend)
- **Filter logic:** `applyNodeFilter` preserves dragged positions via `nodesRef`

## Commands

```bash
npm install
npm run dev      # http://localhost:5173 — landing page → Choose graph.json
npm run build    # output → ../static/erd/ (bundled with Python package)
```
